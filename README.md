**AI Supply Chain Control Tower - Automated Risk Analysis & Nearshoring Agent built with Antigravity AI**

Project Goal
To replace manual supplier monitoring with an autonomous AI Agent that flags logistical risks (OTIF drops, price spikes) and executes mitigation strategies in real-time.

Tech Stack
* AI & Automation: Antigravity (Review-driven AI Agent), Prompt Engineering
* Data: CSV, KPI analysis (OTIF, Price Index)
* Domain Knowledge: Lean Management, Logistics Engineering, Supply Chain Risk Analysis

How It Works (The Workflow)
* Analyzes Data: Evaluates 40 global suppliers based on delivery metrics.
* Flags Risks: Autonomously identifies high-risk/high-cost nodes.
* Executes Solutions: Automatically scans the database for local European alternatives to secure the supply chain and reduce transit time.

Repository Files
*  `updated_suppliers_dataset.csv` - Simulated KPI dataset.
*  `supplier_risk_report.md` - AI-generated risk analysis.
*  `AI_Supply_Chain_Control_Tower.pdf` - Visual documentation & workflow screenshots.
